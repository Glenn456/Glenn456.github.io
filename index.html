Easy
Network Enumeration with Nmap
Hack The Box Academy · Network Enumeration Module
▾
Problem Statement

Perform comprehensive network enumeration on a target range to discover live hosts, open ports, running services, and their versions — simulating real-world reconnaissance as part of a security assessment.

Approach

Started with a host discovery sweep to identify live targets. Followed up with a detailed TCP SYN scan against discovered hosts to enumerate open ports and service banners. Applied OS detection and version scanning flags to build a complete picture of the attack surface. Documented findings in a structured format mimicking a SOC threat intelligence report.

Tools Used
Nmap Netcat Bash scripting Linux CLI

Key Lessons Learned
Service version information is gold for a SOC analyst — it directly maps to CVE databases and helps prioritise which vulnerabilities need urgent attention. Nmap's scripting engine (NSE) dramatically accelerates triage by automating vulnerability checks during enumeration.
Easy
Vulnerability Assessment — Service Identification & Risk Rating
Hack The Box Academy · Vulnerability Assessment Module
▾
Problem Statement

Conduct a structured vulnerability assessment of a target environment — identify weaknesses across the attack surface, assign CVSS risk ratings, and produce a prioritised remediation report.

Approach

Used Nmap to enumerate services, then fed results into OpenVAS for automated vulnerability scanning. Cross-referenced findings against NVD/CVE databases to validate severity. Mapped each vulnerability to the MITRE ATT&CK framework to understand potential attacker TTPs. Produced a risk-prioritised report separating critical/high/medium findings.

Tools Used
Nmap OpenVAS NVD / CVE Database MITRE ATT&CK CVSS Scoring
Screenshot
[ Add your vulnerability scan results screenshot here ]
Key Lessons Learned
Not all vulnerabilities are equal — CVSS scoring alone isn't enough. Context matters: a critical CVE on an internet-facing service is far more urgent than the same finding on an isolated internal host. Effective prioritisation is one of the most valuable skills a SOC analyst can develop.

Use the Metasploit Framework to exploit a known vulnerability on a target machine, establish a Meterpreter session, and perform basic post-exploitation enumeration — understanding attacker tradecraft from a defender's perspective.

Approach

Searched the Metasploit module library for exploits matching identified service versions. Configured the selected module with the target's IP and port parameters. Documented each step to understand what logs and artefacts the attack would generate for a SOC analyst to detect.
